# Study Room App

A real-time collaborative study platform built with React, Node.js, MongoDB, and Socket.IO.

## Live Demo

https://study-room-app-eight.vercel.app

## Features

* Create study rooms
* Join rooms using room ID
* Password-protected rooms
* Real-time chat
* Collaborative whiteboard
* Shared notes
* Online user tracking
* Room owner controls
* Dark mode support

## Tech Stack

### Frontend

* React
* React Router
* Socket.IO Client
* React Sketch Canvas
* Vite

### Backend

* Node.js
* Express
* Socket.IO
* MongoDB
* Mongoose

### Deployment

* Vercel
* Render
